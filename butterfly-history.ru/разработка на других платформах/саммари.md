Яндекс Игры
Коротко: сложно, но реально с доработкой.

Яндекс Игры требуют, чтобы в корень архива добавлен файл index.html, а все файлы весили не более 100 МБ в разархивированном виде. Обязательное требование — встроить SDK Яндекс Игр, и для запуска и использования не требуется регистрация или авторизация на сторонних сервисах.

Главный блокер: Яндекс загружает файлы на свои сервера и запускает как статику. Твой Node.js-бэкенд там не поднимешь. Нужно либо:

переписать так, чтобы фронт сам обращался к Gemini API (но тогда ключ будет торчать в коде — плохо), или
вынести бэкенд на внешний хостинг и обращаться к нему из клиентской части через fetch.
Основное требование платформы — игра как веб-приложение в браузере, предпочтительный формат HTML5. У Яндекс Игр есть категория WebApps, не только игры, где есть дополнительные требования: WebApp не должен выдаваться за официальный продукт внешнего приложения.

Платформа предлагает монетизацию через покупки, подписки и рекламу, использует форматы: полноэкранная реклама, rewarded video и sticky-баннеры. Также есть возможность интегрироваться с другими рекламными сетями через внешнюю монетизацию.

VK Mini Apps (ВКонтакте + Одноклассники)
Коротко: наиболее реалистичный вариант после Telegram.

VK Mini Apps — платформа для мини-приложений внутри ВКонтакте. Можно портировать любое веб-приложение, чтобы получить доступ к огромной аудитории и новым возможностям монетизации.

Важное преимущество: VK Mini Apps можно запустить с внешнего URL — то есть твой бэкенд остаётся на своём хостинге, а VK просто открывает его в iframe/webview внутри соцсети. Нужна лишь интеграция VK Bridge SDK (несколько строк JS).

Причина: это не «игра» с геймплеем, очками, уровнями и retention. Платформы требуют либо чистый HTML5-архив без backend, либо iframe + SDK.
Но да после минимальной доработки (отделить backend на отдельный домен api.butterfly-history.ru с CORS).

4. Гибридная модель монетизации (ARPU > стоимость LLM-токенов)
Обязательная база (чтобы не уйти в минус):

Freemium: 3 бесплатные генерации в день → дальше лимит.
Rewarded Video (30 сек) — разблокирует премиум-модификаторы («Мрачная хроника», «Безумие», изображения).
Interstitial — показывать во время ожидания ответа ИИ (маскирует задержку).
Платные механики (высокий LTV):

Пакеты «ходов» (100 генераций) или «Pro-подписка» (безлимит + приоритет + длинные сценарии).
Telegram Stars / VK Голоса / Яндекс IAP (серверная валидация signed: true).
Отключение рекламы навсегда за разовую покупку.
Дополнительные каналы дохода:

Донаты (Boosty / VK Donut).
B2B: white-label для школ/медиа (образовательный симулятор).
Контент-маркетинг: YouTube/TikTok Shorts «Что если…» → трафик + мерч.
5. Оптимизация затрат на ИИ (чтобы экономика не схлопнулась)
Семантическое кэширование (Redis + embeddings) — до 60 % запросов без вызова LLM.
Prompt Caching (OpenAI/Anthropic/YandexGPT) — расширить системный промпт до >1024 токенов и получить скидку 50–90 %.
Премодерация (wordfilter) — отсекаем токсичные запросы до отправки в LLM.
Серверный middleware — ключ никогда не попадает в браузер.
6. Дорожная карта (реалистичная, 2–4 недели до первых денег)
Неделя 1 (безопасность + базовая монетизация):

Ротация ключа + перенос на сервер.
Freemium + счётчик запросов + rewarded-заглушка.
Запуск Telegram Mini App (бот + WebApp SDK).
Неделя 2:

VK Mini Apps + интеграция VK Bridge.
Яндекс Игры: архив + SDK + safeStorage + LoadingAPI.ready().
Неделя 3–4:

CrazyGames / GameDistribution (английская версия).
Семантическое кэширование + премодерация.
Подключение Stars / VK Pay / Яндекс IAP.
Дальше:

Аналитика (Яндекс Метрика уже есть) → A/B-тесты модификаторов.
Контент в соцсетях → вирусный рост.
