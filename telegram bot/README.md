# Бот «Эффект Бабочки» — Telegram

Telegram-бот на основе сайта [butterfly-history.ru](https://butterfly-history.ru).
Пользователь задаёт вопрос «Что если...», выбирает режим — бот отвечает карточкой-картинкой, в которую уже встроен полный текст альтернативной истории.

---

## Как это работает

```
Пользователь пишет вопрос
        │
        ▼
Бот показывает клавиатуру режимов
        │
        ▼
Пользователь выбирает режим
        │
        ├─ есть квота → генерация → PNG-карточка с полным текстом + ветки продолжения
        │
        └─ нет квоты → предложение купить ⭐ Звёзды
```

## Установка

### 1. Создайте бота в Telegram

1. Напишите `@BotFather` в Telegram
2. `/newbot` → придумайте имя
3. Скопируйте токен

### 2. Добавьте токен в `.env`

В корневом `.env` (рядом с `server.mjs`) добавьте:

```env
BOT_TOKEN=123456789:AABBCCDDEEFFGGHHIIJJ
```

Или создайте отдельный `telegram bot/.env` — бот проверяет оба места.

### 3. Установите зависимости

```bash
cd "telegram bot"
npm install
```

Зависимость `@resvg/resvg-js` подтягивается из родительского `node_modules` автоматически — отдельно устанавливать не нужно.

### 4. Запустите основной сервер (если ещё не запущен)

```bash
cd ..
npm start
```

### 5. Запустите бота

```bash
cd "telegram bot"
npm start
```

## Переменные окружения

| Переменная          | Описание                                       | По умолчанию              |
|---------------------|------------------------------------------------|---------------------------|
| `BOT_TOKEN`         | Токен от @BotFather (**обязательно**)          | —                         |
| `ADMIN_USER_IDS`    | Telegram ID админов через запятую для `/admin_stats` | —                  |
| `SUPPORT_USERNAME`  | Telegram username для `/support` и `/paysupport` | —                    |
| `SUPPORT_EMAIL`     | Email поддержки для `/support` и `/paysupport` | —                       |
| `SUPPORT_URL`       | Ссылка на поддержку для `/support` и `/paysupport` | —                   |
| `TERMS_URL`         | Ссылка на страницу с полными условиями `/terms` | —                     |
| `BUTTERFLY_API_URL` | URL основного сервера                          | `http://localhost:3000`   |

## Квота и оплата

| Тип        | Количество | Стоимость |
|------------|-----------|-----------|
| Бесплатно  | 3 запроса | —         |
| Пакет S    | 10 запросов | 50 ⭐   |
| Пакет L    | 50 запросов | 200 ⭐  |

Цены меняются в `STAR_PACKAGES` в `bot.mjs`.

Stars payments не требуют отдельного платёжного провайдера — работают нативно через Telegram.


## Примечания

- Бот вызывает основной сервер (`/api/alt-history`) — оба должны работать одновременно.
- Контекст для веток хранится в памяти и сбрасывается при перезапуске бота.
- PNG-карточка генерируется на сервере через `@resvg/resvg-js` (никакого headless-браузера).
- Если генерация картинки не удалась, бот отправит обычный текст как запасной вариант.
